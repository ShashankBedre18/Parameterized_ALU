# Parameterized ALU + Register File + Arithmetic Library

## Project Overview

This project is a professional RTL Design project developed using SystemVerilog.

The goal is to build a reusable parameterized Arithmetic Logic Unit (ALU), Register File, and Arithmetic Library that can later be integrated into larger digital systems such as:

- RISC-V Processor
- Microcontroller
- CPU Datapath
- FPGA Designs
- ASIC Designs

This project follows an industry-style RTL design methodology from specification to verification.

---

## Objectives

- Learn professional RTL coding practices
- Design reusable parameterized modules
- Develop synthesizable SystemVerilog RTL
- Create self-checking testbenches
- Simulate and debug using Vivado
- Document every module
- Version control using Git and GitHub
